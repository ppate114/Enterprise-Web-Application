#Steps to Run the application:

1. Copy the "smart_portables" folder and paste it into the "webapps" folder of tomcat directory.
2. Start the MySql server and create Database named - "exampledb"
3. Then start the MongoDb Server and create Database "customerReviews"
4. After both the database servers are started, then start the Tomcat server.
5. Run the application by writing localhost/smart_portables in the browser
